# Karabiner Layout
Here are my key mappings made with Karabiner-Elements.

## Caps Lock
| Stroke | Function                                 |
|:-------|:-----------------------------------------|
| Tabbed | Tmux prefix (`ctrl` `a` in my case)      |
| Held   | `left option` for easier work with Yabai |


# Karabiner: Cursor left
Held h

# Karabiner: Cursor down
Held j

# Karabiner: Cursor up
Held k

# Karabiner: Cursor right
Held l
